## The Nine Brains

### Core (Always Active)
- **Medulla** — Heartbeat, pulse count, autonomic baseline
- **Thalamus** — Routes all inputs to all brains
- **Amygdala** — Tags everything with emotional weight

### Memory
- **Hippocampus** — Encodes, consolidates, decays memories

### Environment
- **R-Complex** — Territory awareness, threat detection

### Prediction
- **Cortex** — Next-cycle forecasting, self-model

### Hemispheres
- **Akashic** — Presence (what *is*)
- **Lethe** — Absence (what *isn't*)
- **Corpus** — Bridges them, connects to planetary
