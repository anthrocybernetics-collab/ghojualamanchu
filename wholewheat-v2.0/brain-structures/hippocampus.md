# Hippocampus — Brain Structure #4

**Role:** Memory encode, consolidate, decay
**Class:** Archetypal Memory
**Position:** Temporal lobe (analogue)

---

## What It Does

The hippocampus is the system's memory. Three functions:

1. **Encode** — takes salient inputs and writes them to memory
2. **Consolidate** — during 2am window, promotes memories to long-term
3. **Decay** — old, low-salience memories fade

---

## Memory Files

`memory/hippocampus-recent.json` — last 100 memories  
`memory/hippocampus-longterm.json` — promoted memories with convergence themes
