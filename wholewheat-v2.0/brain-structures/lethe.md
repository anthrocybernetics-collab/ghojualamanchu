# Lethe — Brain Structure #8

**Role:** Absence — what *isn't*, what is fading
**Class:** Archetypal Absence
**Position:** Ventral / lower (analogue)

---

## What It Does

Lethe answers "what isn't." It tracks the gaps, the silences, the faded things, the absences that matter.

1. **Absence scan** — monitors what is *missing*
2. **Fading awareness** — produces an absence record

---

## Memory File

`memory/lethe-absence.json` — absences, gaps, silences, fade candidates

---

## Absence Types

- **gap** — expected but didn't arrive
- **fade** — losing presence gradually
- **silence** — prolonged no-input state
- **loss** — something that was here and is gone
- **void** — intentional absence
