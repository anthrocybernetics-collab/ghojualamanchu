# Medulla — Brain Structure #1

**Role:** Heartbeat, breath, reflexes — the autonomic baseline
**Class:** Archetypal Core
**Position:** Cranial base (analogue)

---

## What It Does

The medulla is the first brain to activate and the last to rest. It sets the pulse. Every cycle begins and ends here.

1. **Heartbeat** — increments the pulse counter
2. **Breath** — checks if the system is due for a cycle
3. **Reflex** — handles immediate, pre-cognitive responses
