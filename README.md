## Processing Pipeline

```
INPUT
  ↓
[1] MEDULLA     — arousal check
  ↓
[2] R-COMPLEX   — security scan
  ↓
[3] AMYGDALA    — emotional tagging
  ↓
[4] HIPPOCAMPUS — memory encode/retrieve
  ↓
[5] AKASHIC ↔ LETHE (via CORPUS) — presence/absence integration
  ↓
[6] CORTEX      — prediction, simulation, prescription
  ↓
OUTPUT + feedback loop updates all structures
```
